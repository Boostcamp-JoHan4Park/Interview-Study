## Study Process
매주 아래의 과정을 반복

1. 각자 맡을 분야를 랜덤하게 선정(ML, python, network, etc...)
2. 모든 분야에 대한 답을 준비
3. 본인이 맡은 분야에 대해서는 '면접에서 답변할 수 있는 형태'와 키워드를 정리.
4. 2, 3번의 내용을 본인 이름의 폴더에 자신만의 표현으로 정리 
(e.g., sungho/)
5. 스터디에서 정리된 내용을 answer/*.md에 정리
